---
title: 'Getting started'
---

System Requirements:

- Linux, MacOS or Windows
- NodeJS 8+

To install the command line tool, run the command:

    npm install @studiorack/studiorack-cli -g

Verify the tool has been installed by running:

    studiorack --version
